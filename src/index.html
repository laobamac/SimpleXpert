SSDT热补丁定制

根据您的硬件配置生成适合的SSDT热补丁，用于解决黑苹果系统的ACPI问题。

SSDT补丁列表
SSDT-PLUG CPU电源管理
SSDT-AWAC 系统时钟修复
SSDT-EC-USBX 嵌入式控制器
SSDT-PMC NVRAM支持
生成的SSDT代码
/*
 * Intel ACPI Component Architecture
 * AML/ASL+ Disassembler version 20200925 (64-bit version)
 * ...
 */

DefinitionBlock ("", "SSDT", 2, "ACDT", "CPUPLUG", 0x00000000)
{
    External (_SB_.PCI0, DeviceObj)
    External (_SB_.PCI0.LPCB, DeviceObj)
    
    Scope (\_SB.PCI0.LPCB)
    {
        Device (EC)
        {
            Name (_HID, "ACPI000D")  // _HID: Hardware ID
            Name (_UID, One)  // _UID: Unique ID
            Name (_GPE, 0x1B)  // _GPE: General Purpose Events
            Method (_REG, 2, NotSerialized)  // _REG: Region Availability
            {
                If (_OSI ("Darwin"))
                {
                    Store (0x0D, DBG8)
                }
            }
        }
    }
    
    Scope (\)
    {
        Method (DTGP, 5, NotSerialized)
        {
            If ((Arg0 == ToUUID ("a0b5b7c6-1318-441c-b0c9-fe695eaf949b")))
            {
                If ((Arg1 == One))
                {
                    If ((Arg2 == Zero))
                    {
                        Store (Buffer (One) { 0x03 }, Arg4)
                        Return (One)
                    }
                    
                    If ((Arg2 == One))
                    {
                        Return (One)
                    }
                }
            }
            
            Store (Buffer (One) { 0x00 }, Arg4)
            Return (Zero)
        }
    }
}

SimpleXpert
版本 1.0.0

SimpleXpert 是一款专业的黑苹果(Hackintosh)工具集，旨在简化黑苹果安装和配置过程。

主要功能

  • SSDT热补丁自动生成
  • USB端口定制工具
  • macOS镜像下载
  • 多种黑苹果辅助工具

系统要求

  • Windows 10/11 64位
  • macOS 11.0+ (用于部分功能)
  • 4GB 内存
  • 800MB 可用磁盘空间

开发者

SimpleXpert 由黑苹果爱好者社区开发维护。
